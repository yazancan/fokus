# Fokus

Aplicação web desenvolvida com foco em produtividade, utilizando a técnica Pomodoro para auxiliar na gestão de tempo e foco durante tarefas.

## 🚀 Tecnologias

- HTML
- CSS
- JavaScript

## 💡 Funcionalidades

- Temporizador com técnica Pomodoro
- Alternância entre modos (foco, descanso curto e descanso longo)
- Interface dinâmica com mudança de contexto visual
- Reprodução de áudio ambiente para auxiliar na concentração

## 🧠 Aprendizados

- Manipulação do DOM com JavaScript
- Controle de tempo com funções de temporização
- Organização de estados da aplicação (foco/descanso)
- Atualização dinâmica de elementos na interface

## ✨ Destaque

Projeto desenvolvido durante a formação de front-end da Alura, com aplicação prática de JavaScript para criação de uma aplicação interativa e funcional.
